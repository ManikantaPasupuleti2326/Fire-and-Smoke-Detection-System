# 🔥 Fire and Smoke Detection System

A smart **Fire and Smoke Detection System** developed using **Arduino Uno**, **MQ-2 Smoke Sensor**, and **Flame Sensor** to detect fire and smoke in real time. The system provides immediate alerts through a buzzer and LED indicators, helping prevent accidents and improve safety in homes, offices, laboratories, and industrial environments.

---

# 📖 Project Overview

Fire accidents can cause severe damage to life and property if not detected early. This project continuously monitors the surrounding environment using smoke and flame sensors. When smoke concentration exceeds a predefined threshold or a flame is detected, the system activates an alarm and visual indicators, enabling quick response to potential fire hazards.

# ⚙️ System Workflow

1. The MQ-2 sensor continuously monitors smoke and combustible gases.
2. The flame sensor detects the presence of fire.
3. The Arduino Uno reads sensor values.
4. If smoke or fire is detected above the threshold:

   * The buzzer is activated.
   * LED indicators turn ON.
5. The alarm remains active until the hazardous condition is cleared.

---

# 🧠 Working Principle

The system continuously receives input from the MQ-2 smoke sensor and the flame sensor. The Arduino processes these sensor readings and compares them with predefined threshold values. If either smoke or flame is detected, the controller immediately triggers the buzzer and LED indicators to alert nearby people, enabling rapid action to minimize potential damage.
